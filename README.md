# Mimosa-Flytrap

Private AI agent projects, A curated repository that aggregates and organizes AI Agent-related scripts, installation & configuration documents, and runtime example documents for programming, document processing, image processing, and other tasks across diverse environments.

## Directory Structure

The repository is organized by operating system at the top level:

- **[macOS](./macOS/)** - AI Agent resources for macOS
- **[linux-ubuntu-22.04-lts](./linux-ubuntu-22.04-lts/)** - AI Agent resources for Linux Ubuntu 22.04 LTS
- **[windows-10-11](./windows-10-11/)** - AI Agent resources for Windows 10/11

Each directory contains scripts, installation & configuration documents, and runtime examples specific to that operating system.
